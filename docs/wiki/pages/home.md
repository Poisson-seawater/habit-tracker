# Habit Tracker RPG

Un habit tracker RPG auto-hébergé qui transforme la discipline quotidienne en jeu — XP, niveaux, or, streaks, journées parfaites — pour se tenir mutuellement responsable de ses habitudes entre amis. Bot Telegram, dashboard web ou skills LLM, une même base SQLite locale sur Raspberry Pi 5. Pas un SaaS : tout est personnalisé pour un usage perso.

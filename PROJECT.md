| Field | Value |

| Type | Personal app / MVP |

## Goal

Build a simple habit tracking app first, then decide whether it should become a richer web app with accounts, sync, reminders, and analytics.

## Current Scope

- Local browser prototype.
- Habit creation and deletion.
- Daily check-off.
- Streak and weekly progress stats.
- No backend yet.
